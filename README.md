# 2,500 Years of Wellness

**The Yellow Emperor's Guide to Living Well**

**养生两千五百年：黄帝内经的生活智慧**

---

## About

This book reinterprets the *Huangdi Neijing* (黄帝内经) — a 2,500-year-old Chinese medical classic — as a modern wellness lifestyle guide. Each chapter bridges ancient wisdom with contemporary science, providing actionable daily practices backed by both tradition and research.

- **Bilingual**: Chinese (zh) + English (en) parallel editions
- **9 chapters** + Preface, covering rhythm, food, emotions, movement, prevention, balance, sleep, and a 90-day integration plan
- **Evidence-graded**: Every Neijing principle is tagged ✓ (confirmed), ? (plausible hypothesis), or ✗ (disproven)
